The special skills Goku has

Energy Ball: This is an attack that only Goku knows how to do. You have to be a very special person to be able to do this attack because you need to know how to collect energy from many different places.




Rapid transformation: this is an ability that Goku learned from very special people, this ability allows him to move to any place he wants in a few seconds, this is something that only he knows how to do from the warrior race

Ultra Instinct: This is one of the rarest abilities there is and almost no living creature has been able to acquire this ability except for Goku who was able to obtain it under great pressure placed on him, this ability allows him to move very, very fast
